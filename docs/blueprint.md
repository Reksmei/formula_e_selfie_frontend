# **App Name**: E-Prix Imagery

## Core Features:

- Camera Integration: Access the device's camera to capture a selfie.
- Prompt Selection: Allow users to select from a list of Formula E-related prompts.
- Image Generation: Generate a new image based on the user's selfie and the selected prompt using a generative AI tool. It uses the image and the prompt as inputs, then it may or may not use them to generate a new image based on them.
- Image Display: Display the original selfie and the generated image side-by-side.
- Error Handling: Display error messages for camera access issues or failed image generation.

## Style Guidelines:

- Primary color: Electric Blue (#007BFF) to represent the energy and speed of Formula E racing.
- Background color: Dark gray (#222222) for contrast, allowing the images and text to pop.
- Accent color: Bright Yellow (#FFD700) for interactive elements like buttons and highlights, providing a visual cue.
- Font: 'Inter', sans-serif, for a modern, clean, and readable user interface.
- Note: currently only Google Fonts are supported.
- Use a set of modern, minimalist icons to represent different features.
- A centered layout to focus on the image generation process.
- Use subtle loading animations to indicate image generation progress.
